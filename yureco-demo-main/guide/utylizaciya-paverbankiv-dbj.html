Призначення довідки

Практичні орієнтири для підприємств: що відноситься до цієї категорії, типові ризики Li-ion/LiPo, як підготувати партію до передачі та які документи потрібні для підтвердження утилізації.

Логіка матеріалу для внутрішніх регламентів і швидких погоджень: визначення → ризики → що приймають → підготовка → процес → документи → FAQ.

Визначення

  • Що це: павербанки, ДБЖ (UPS), акумуляторні блоки та компоненти енергоживлення, що підлягають передачі на переробку/утилізацію/знешкодження з урахуванням вимог безпеки та обліку.
  • Де виникає: списання парку пристроїв, оновлення інфраструктури, поломки, брак, повернення, заміна батарейних модулів.

Ризики

  • Підвищений пожежний ризик через пошкоджені або здуті Li-ion/LiPo елементи.
  • Коротке замикання при контакті клем/роз’ємів без ізоляції.
  • Відмова у прийманні через змішування фракцій, відсутність маркування або “невідомий склад”.
  • Ризики для обліку та перевірок у разі відсутності підтвердних документів.

Що зазвичай приймають / що можуть не приймати

  • Приймають: павербанки, ДБЖ, акумуляторні модулі/блоки, окремі батареї (за погодженням), відсортовані та упаковані партії.
  • Можуть не приймати: пристрої з протіканнями, сильно пошкоджені/“невідомі” батареї без опису, партії зі змішаними фракціями, розкриті/розсипані елементи без тари.

Як підготувати до передачі

  1. Відсортувати: окремо павербанки, окремо ДБЖ/UPS, окремо батарейні модулі/елементи (за можливості).
  2. Ізолювати контакти (клеми/роз’єми) для запобігання короткому замиканню.
  3. Упакувати в міцну тару з фіксацією, щоб уникнути ударів і переміщення під час транспортування.
  4. Окремо виділити пошкоджені/здуті елементи та позначити це в описі.
  5. Промаркувати: тип (павербанк/UPS), орієнтовна хімія (Li-ion/LiPo — якщо відомо), стан, контакт відповідальної особи.
  6. Підготувати короткий опис і 3–5 фото партії для погодження.

Типовий процес утилізації

  1. Узгодження складу/обсягу, вимог до пакування та логістики.
  2. Передача партії (вивезення/приймання).
  3. Сортування: електроніка/пластик/метал + окремо батарейні компоненти.
  4. Демонтаж (за потреби) та направлення фракцій на переробку або знешкодження.
  5. Видача документів.

Документи (типово)

  • акт приймання-передачі;
  • акт виконаних робіт / акт утилізації (підтвердження для обліку);
  • за потреби: специфікація/перелік позицій, фотофіксація (за домовленістю).

FAQ

  • Як зменшити витрати?
    Сортування, правильна тара, ізоляція контактів і компактне складування знижують ризики та спрощують оброблення партії.
  • Що найчастіше “ламає” приймання?
    Змішування фракцій, відсутність маркування, розсипані елементи без тари, пошкоджені батареї без позначення стану.
